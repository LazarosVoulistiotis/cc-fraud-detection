# Dataset Overview

- **Αρχείο**: `data/data_raw/creditcard.csv`  
- **Μέγεθος**: **284,807 γραμμές × 31 στήλες**  
- **Fraud rate**: **0.173%** (492 / 284,807)

---

## 1. Περιγραφή πεδίων

- **Time**  
  Μετράει δευτερόλεπτα από την πρώτη συναλλαγή στο dataset.  
  Χρησιμοποιείται για να μελετηθεί η χρονική ακολουθία συναλλαγών, αλλά δεν αντιστοιχεί σε πραγματική ημερομηνία.

- **V1 – V28**  
  Ανώνυμα αριθμητικά χαρακτηριστικά που προέκυψαν με **PCA (Principal Component Analysis)** για προστασία απορρήτου.  
  Δεν έχουν άμεση ερμηνεία, αλλά κωδικοποιούν συνδυασμούς αρχικών χαρακτηριστικών.

- **Amount**  
  Το χρηματικό ποσό κάθε συναλλαγής (€).  
  Συνήθως απαιτεί κανονικοποίηση (scaling) πριν χρησιμοποιηθεί σε μοντέλα.

- **Class**  
  Ετικέτα στόχου:  
  - `0` = κανονική συναλλαγή (Legit)  
  - `1` = απάτη (Fraud)
